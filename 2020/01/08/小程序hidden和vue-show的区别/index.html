小程序 hidden 工作原理

小程序添加了 hidden 的标签，底层是通过选择器的方式实现隐藏。

底层通过 view[hidden]{ display: none; } 属性选择器，实现的 div 隐藏。

既然涉及到了选择器，就会受到权重的影响，行内样式权重过高，会导致 hidden 隐藏无效。

  1. 添加了 hidden 的盒子，匹配了内置的选择器，实现隐藏。
  2. 如果 hidden 条件为 true，相当于 display: none 生效。
  3. 如果 hidden 条件为 false，属性没有了，所以 display 用回原本的值。

Vue v-show 原理

v-show 是通过行内样式修改了盒子的显示与隐藏，行内样式权限高，问题较少。

  1. 添加了 v-show 的盒子，先检查盒子本身的 display 属性的值，存起来。
  2. 如果条件为 true，就把属性值设置了 display 之前存起来的值。
  3. 如果条件为 false，就把属性值设置了 display: none;
